ODCS-FusionTool
========

ODCS-FusionTool is a standalone tool for execution of data fusion and conflict resolution on RDF data accessible through a SPARQL endpoint. ODCS-FusionTool is based on the conflict resolution component implemented in <a href="https://sourceforge.net/p/odcleanstore/wiki/Home/">ODCleanStore</a>, a Linked Data management tool.
